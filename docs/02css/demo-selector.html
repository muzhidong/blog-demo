xxx-child与xxx-of-type选择器的最大区别在于，
前者要匹配指定选择器，也要匹配位置，而后者只需要匹配指定选择器，
另外，only-child与only-of-type稍微不同，两者都要求匹配选择器，但是前者要求只能有一个子元素，后者要求只能有一个匹配该选择器的子元素。
